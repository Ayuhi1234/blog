A side project born from wanting a GitHub-contributions-style heatmap that worked across private repos and self-hosted git servers without sending any data anywhere. Built in Rust mostly as an excuse to get more comfortable with the language.

## Status

Core heatmap rendering and local SQLite indexing work. The weekly digest emailer is half-built — the templating works, the scheduler doesn't yet.
